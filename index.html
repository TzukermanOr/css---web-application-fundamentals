מה זה CSS?

Cascading Style Sheets היא שפה שאחראית על המראה של דפי אינטרנט. HTML = מבנה (שלד), CSS = עיצוב (עור ובגדים).

Cascading = מדורג - כשיש כמה חוקים על אותו אלמנט, הדפדפן מחליט מי מנצח לפי סדר עדיפויות (Specificity).

הנקודה המרכזית: CSS מפריד בין תוכן לעיצוב. אתה יכול לשנות את כל המראה של אתר שלם ע"י עריכת קובץ CSS אחד בלבד, בלי לגעת ב-HTML.

התחביר הבסיסי

כל חוק CSS בנוי מ-3 חלקים: סלקטור (מי), תכונה (מה), ערך (איך).

CSS /* הסלקטור בוחר את האלמנט */ body { background-color: lightblue; } h1 { color: navy; /* צבע טקסט */ margin-left: 20px; /* מרווח שמאלי */ } p { text-decoration: underline; }

יתרונות CSS

Grouping - עיצוב קבוצתי

אפשר לעצב כמה אלמנטים בו-זמנית. למשל, כל הפסקאות באתר בגופן אחיד:

CSS /* כל הפסקאות + כל הכותרות בבת אחת */ p, h1, h2, h3 { font-family: "Segoe UI", sans-serif; }

Avoid Duplication - מניעת כפילות

קובץ CSS אחד משרת כמה דפי HTML. שינוי אחד = שינוי בכל האתר.

Clean Code - קוד נקי

הפרדה בין HTML (מבנה) ל-CSS (עיצוב). קל יותר לקרוא, לתחזק ולשנות.

HTML vs CSS - עיצוב טקסט

ב-HTML יש תגיות עיצוב בסיסיות. ב-CSS אפשר להשיג את אותו דבר בצורה טובה יותר:

HTML way <b>Bold text</b> <i>Italic text</i> <u>Underline</u> <mark>Highlight</mark> <small>Small</small>
CSS way (עדיף!) element { font-weight: bold; } element { font-style: italic; } element { text-decoration: underline; } element { background-color: yellow; } element { font-size: smaller; }

Specificity: 1000לא מומלץ

משימה 1 - בסיסית

יצירת קובץ HTML + קובץ CSS עם כל 3 השיטות, ID, Class, hover ו-active

דרישות
  1. צרו קובץ index.html וקובץ style.css
  2. הוסיפו 3 כותרות: h1, h2, h3 - כל אחת מעוצבת אחרת

  5. הוסיפו :hover (שינוי צבע במעבר עכבר)
  6. הוסיפו :active (שינוי צבע בלחיצה)
  7. הוסיפו טופס login בסיסי
