Bootstrap Navbar - הbar שלמעלה

Bootstrap

ה-navbar למעלה בנוי עם Bootstrap 5. הוא מגיב - מתקפל לתפריט המבורגר במסכים קטנים.

פירוק ה-classes העיקריים:

  • navbarclass בסיסי - מפעיל את רכיב הnavbar
  • navbar-expand-lgמתפרש אופקית ב-lg (1024px+), מתקפל מתחת
  • navbar-brandהלוגו/שם האתר בצד שמאל
  • navbar-togglerכפתור ה"המבורגר" - מופיע רק במסכים קטנים
  • collapse navbar-collapseהעטיפה של הלינקים - נסתרת/מוצגת
  • nav-item / nav-linkכל פריט ולינק בתפריט
  • dropdownתפריט נפתח - דורש JS של Bootstrap
  • me-automargin-end: auto - דוחף את החיפוש לצד ימין
חשוב: Bootstrap's JS נטען בתחתית הdocument.
בלי זה - dropdown ו-toggler לא יעבדו!
השוואה: Navbar שלך VS Bootstrap
הקוד שלך:
CSS ידני - כתבת padding, gap, color לבד
Bootstrap:
classes מוכנות - Bootstrap כותב את ה-CSS בשבילך
מה Bootstrap עושה מאחורי הקלעים
.navbar-expand-lg .navbar-nav ↳ flex-direction: row; justify-content: flex-start .me-auto ↳ margin-inline-end: auto !important .collapse:not(.show) ↳ display: none
מבנה Navbar בסיסי
<!-- טעינת Bootstrap CSS ב-head -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">

<nav class="navbar navbar-expand-lg">
  <div class="container-fluid">
    <a class="navbar-brand" href="#">שם האתר</a>
    <!-- כפתור המבורגר למובייל -->
    <button class="navbar-toggler" data-bs-toggle="collapse" data-bs-target="#nav">
      <span class="navbar-toggler-icon"></span>
    </button>
    <div class="collapse navbar-collapse" id="nav">
      <ul class="navbar-nav me-auto">
        <li class="nav-item"><a class="nav-link active" href="#">בית</a></li>
      </ul>
    </div>
  </div>
</nav>

<!-- JS של Bootstrap - חובה בתחתית ה-body! -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

Inline / Internal / External CSS

יסודי

יש 3 דרכים להוסיף CSS לדף:

  • Inlineבתוך תג HTML עצמו - style=""
  • Internalבתוך תג <style> ב-<head>
  • Externalקובץ .css נפרד, מקושר עם <link>
סדר עדיפות (גבוה לנמוך):
Inline (1000) > Internal/External (לפי סדר הופעה)

מה מומלץ? External לכמעט הכל.
Internal - לדף בודד. Inline - רק במקרי קצה.
מה לשים לב: Inline עוקף כמעט הכל - קשה לתחזק. External מאפשר לשתף CSS בין דפים.

זה Inline CSS - style="" בתוך התג

style="color: coral; font-weight: bold"

3 הדרכים - קוד מלא
/* 1. INLINE - ישירות על התג */
<p style="color: coral; font-weight: bold;">טקסט</p>

/* 2. INTERNAL - בתוך head */
<head>
  <style>
    p { color: coral; }
  </style>
</head>

/* 3. EXTERNAL - קובץ נפרד style.css */
<head>
  <link rel="stylesheet" href="style.css">
</head>
/* ובקובץ style.css: */
p { color: coral; }

display - בלוקים ויחידות תצוגה

יסודי

display קובע איך אלמנט תופס מקום בדף:

  • blockתופס שורה שלמה, אפשר להגדיר width/height. דוג: div, p, h1
  • inlineרק כמה שצריך לתוכן, לא מכבד width/height. דוג: span, a, strong
  • inline-blockזורם כמו inline אבל מכבד width/height
  • flexמפעיל Flexbox על הילדים
  • gridמפעיל Grid על הילדים
  • noneמסתיר לחלוטין - לא תופס מקום

block - כל אחד בשורה משלו
display: block - תופס שורה שלמה
גם זה block - מתחיל שורה חדשה
inline - זורמים יחד בשורה
inline inline גם inline
inline-block - זורם אבל מכבד גודל

לחצנים - Buttons

ביניים

לחצן טוב צריך: מצב רגיל, hover, active, ו-focus לנגישות.

  • cursor: pointerהופך את הסמן ליד - חובה!
  • border: noneמסיר border ברירת מחדל של הדפדפן
  • transitionמעבר חלק ב-hover
  • :hoverמצב hover
  • :activeברגע הלחיצה
  • :focusכשנבחר עם Tab - לנגישות!
  • :disabledמצב מנוטרל
  • outline: noneמסיר outline אוטומטי - זהירות: נגישות!
אל תסיר outline ממוקד - זה פוגע בנגישות (משתמשי Tab). אם מסיר, צור עיצוב focus אלטרנטיבי.
סוגי לחצנים - עבור עם העכבר!

filter - פילטרים ויזואליים

ביניים

filter מוסיף אפקטים ויזואליים לאלמנט - כמו Photoshop ב-CSS:

  • blur(px)טשטוש
  • brightness(%)בהירות. 1=רגיל, 2=כפול, 0=שחור
  • contrast(%)ניגודיות
  • grayscale(%)גווני אפור. 1 = שחור לבן מלא
  • hue-rotate(deg)סיבוב גלגל הצבעים
  • invert(%)היפוך צבעים
  • saturate(%)רוויית צבע. 0=אפור, 3=חי מאוד
  • sepia(%)אפקט ישן/ספיה
  • drop-shadow()צל שמכבד צורת הפיקסלים (שונה מbox-shadow)
אפשר לשלב כמה פילטרים:
filter: blur(2px) brightness(1.5);

קוד
img { filter: grayscale(1); }               /* שחור לבן */

.dark-overlay { filter: brightness(0.6); }    /* להכהות תמונה */

מיקום בדף - Position & z-index

ביניים
  • staticברירת מחדל - זרימה רגילה
  • relativeמוזז מהמיקום הרגיל. הפינה "עוגן" לילדים absolute
  • absoluteמוצא מהזרימה. ממוקם יחסי לאב relative הקרוב
  • fixedנשאר במסך גם בגלילה. יחסי לViewport
  • stickyזורם עד נקודה, ואז נדבק - כמו navbar שלנו!
  • top/right/bottom/leftקובע מיקום (רק ל-non-static)
  • z-indexשכבת עומק - מי מעל מי. גבוה יותר = מעל
z-index עובד רק על אלמנטים עם position שאינו static!
overflow - מה קורה כשתוכן גולש:
visible (ברירת מחדל) | hidden | scroll | auto
absolute בתוך relative
static relative (top:5 right:5) absolute (bottom:12 left:12) fixed (בד"כ לViewport)
z-index - שכבות
z:1
z:2
z:3 (מעל)
קוד

.modal {
  position: fixed;
  top: 50%; left: 50%;

/* JavaScript */
const canvas = document.getElementById('myCanvas');
const ctx = canvas.getContext('2d');

/* מלבן */
ctx.fillStyle = '#4f8ef7';
ctx.fillRect(20, 20, 100, 60);                           /* x, y, width, height */

/* עיגול */
ctx.beginPath();
ctx.arc(200, 100, 40, 0, Math.PI * 2);                /* x,y,radius,start,end */
ctx.fillStyle = '#e05c7c';
ctx.fill();

/* טקסט */
ctx.fillStyle = 'white';
ctx.font = 'bold 20px Arial';
ctx.fillText('שלום Canvas!', 20, 140);
